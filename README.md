## 🧬 نظرة عامة (Overview)
أداة **DAJ-STRIKE** هي حقيبة برمجية عالية الأداء مصممة خصيصاً لمهندسي الشبكات وفنيي الصيانة الميدانية وأخصائيي الفحص السيبراني. تتميز بواجهة سايبرانية احترافية بألوان النيون، وتقوم بمسح الشبكة ذاتياً، وعزل أجهزة الراوتر الأساسية والإكسس بوينت (APs) المخفية، وسحب الأسماء والأنواع الحقيقية للأجهزة المتصلة من أنظمتها مباشرة دون قيود.

---

## 🚀 الميزات الرئيسية (Key Features)
* **رسم بنية الشبكة (Map Network Architecture):** كشف وعزل أجهزة الراوتر والـ Access Point عبر فحص ذكي للمنافذ المفتوحة (80/443).
* **سحب الأسماء الحقيقية (Deep Name Resolution):** فحص موازٍ فائق السرعة يسحب موديلات الهواتف والأجهزة المتصلة (مثل Redmi, Galaxy) عبر بروتوكولات mDNS و DHCP.
* **مراقب استقرار الإشارة (Signal Stability Monitor):** بث حي ومفلتر لحزم الاتصال (Ping) لقياس زمن الاستجابة بالملي ثانية (ms) ومعرفة الجودة الفعلية للخط.
* **الفحص الآمن (Safe-Thread Execution):** برمجة متطورة ومتوافقة مع معالجات الهواتف لمنع تجميد النظام أو إغلاق التطبيق المفاجئ (`Signal 9`).

## 💻 الأنظمة المدعومة (Compatibility)
الأداة مبرمجة بلغة **Bash Script** وتعمل على جميع الأنظمة المبنية على بيئة Unix/Linux:
* **تطبيق Termux (أندرويد):** مستقر تماماً ومعدل لتخطي حماية الأندرويد الحديثة بدون روت.
* **أنظمة Kali Linux / Parrot OS / Ubuntu:** يعمل بشكل أصلي ومثالي لفحص الشبكات المحمول.
* **نظام Windows:** مدعوم فقط عبر بيئة **WSL** (Windows Subsystem for Linux).

## 🛠️ طريقة التثبيت والتشغيل السريع (Installation)

افتح مبوب الأوامر (Terminal) في نظامك وانسخ أمر السطر الواحد التلقائي (One-Line Installer) المخصص لنظامك:

### 1. للتثبيت على تطبيق ترمكس (Termux - Android):
```bash
pkg install git traceroute nmap dnsutils -y && git clone https://github.com ~/daj-tmp && mv ~/daj-tmp/scan ~/find-ap && chmod +x ~/find-ap && ln -sf ~/find-ap \$PREFIX/bin/scan && rm -rf ~/daj-tmp && scan
```

### 2. للتثبيت على نظام كالي لينكس أو التوزيعات الأخرى (Kali Linux / Linux):
```bash
sudo apt update && sudo apt install git traceroute nmap dnsutils -y && git clone https://github.com ~/daj-tmp && sudo mv ~/daj-tmp/scan /usr/local/bin/scan && chmod +x /usr/local/bin/scan && rm -rf ~/daj-tmp && scan
```

## ⚙️ تعليمات الاستخدام (Usage)
بعد تثبيت الأداة، يمكنك تشغيلها في أي وقت ومن أي مسار داخل هاتف الصيانة أو النظام بكتابة هذه الكلمة فقط:
```bash
scan
```
ثم تنقل عبر القائمة التفاعلية:
* اضغط رقم `1` لعزل الراوترات والإكسس بوينت (APs).
* اضغط رقم `2` لسحب آيبيهات وأسماء وموديلات الأجهزة المتصلة.
* اضغط رقم `3` لمراقبة استقرار وسرعة استجابة خط الإنترنت حياً.
