# ⚡ DAJ - STRIKE SUITE v2.0 ⚡

أداة متقدمة وخفيفة الوزن وعالية الأداء لتحليل الشبكات وفحص الأمان الرقمي، تم تصميمها وهندستها خصيصاً لتعمل بسلاسة فائقة داخل بيئة **Termux**. تم بناء الأداة مع التركيز على كفاءة الأداء، حيث تدمج بين سرعة معالجة سكربتات باش (Bash) وقوة تعدد المسارات (Multithreading) في لغة بايثون لتقديم استطلاع سريع للبنية التحتية وتحليل استقرار الشبكة.

## 🎛️ موديلات النظام والميزات (System Modules)

| رقم الموديل | اسم الموديل بالعربية | التقنية / المحرك المستخدم | وصف الموديل البرمجي |
| :---: | :--- | :--- | :--- |
| **01** | **خريطة البنية التحتية** | `nmap` + `netcat` | رسم خريطة للموجهات والموزعات (Routers & Access Points) عبر تشغيل متوازٍ في الخلفية. |
| **02** | **سحب هويات الأجهزة** | `nmap (Host Detection)` | اكتشاف وفحص جميع الأجهزة المتصلة (هواتف وحواسيب) على شبكة الـ `/24` الفرعية. |
| **03** | **قناص المنافذ** | `nmap -T4 -F` | فحص أمان المنافذ والأنظمة بشكل سريع جداً لتحديد الخدمات المفتوحة والنشطة. |
| **04** | **استخبارات الشبكة الخارجية**| `curl` + `ip-api.com` | الاستعلام الفوري عبر الـ API لجلب الـ Public IP، الموقع الجغرافي، واسم مزود الإنترنت (ISP). |
| **05** | **اختبار سرعة الـ DNS** | `ping (ICMP Matrix)` | قياس دقيق ومقارنة حية لزمن الاستجابة بين مخدمات الـ DNS العالمية (Google, Cloudflare, Quad9, OpenDNS). |
| **06** | **مراقب استقرار الاتصال** | `ping (Reactive Loop)` | مراقبة حية لدفق حزم البيانات مع تنبيهات لونية تلقائية (أخضر/أصفر/أحمر) عند تدهور الشبكة. |
| **07** | **اختبار جهد الشبكة** | `python3 (UDP Threading)`| تقييم جدران الحماية عبر إغراق حزم ICMP أو عبر محرك بايثون سريع جداً مخصص لإغراق حزم UDP. |

## 🚀 أمر التثبيت التلقائي السريع (بضغطة واحدة)

يمكنك تشغيل الحزمة واختصار أمرها فوراً. افتح تطبيق الترمكس، وانسخ الأمر التالي كاملاً لتبدأ الأداة بالعمل:

```bash
pkg install git -y && git clone https://github.com && cd YOUR_REPOSITORY_NAME && mv daj.sh \(PREFIX/bin/daj && chmod +x\)PREFIX/bin/daj && daj
```

### 🛠️ الإعداد اليدوي خطوة بخطوة
إذا كنت تفضل تثبيتها وضبط اختصار الأمر يدوياً:

1. **انسخ المستودع إلى جهازك:**
   ```bash
   git clone https://github.com
   cd YOUR_REPOSITORY_NAME
   ```

2. **انقل السكربت إلى مجلد باينري النظام مع إزالة اللاحقة (.sh):**
   ```bash
   mv daj.sh \$PREFIX/bin/daj
   ```

3. **امنح الملف صلاحيات التنفيذ الشاملة كامر نظام:**
   ```bash
   chmod +x \$PREFIX/bin/daj
   ```

4. **شغّل الأداة من أي مسار أو مجلد مستقبلاً بكلمة واحدة:**
   ```bash
   daj
   ```

## 🧰 التثبيت الذكي للمتطلبات (Auto-Installer)
تتميز الأداة بوجود نظام إدارة حزم مدمج تلقائي. عند أول تشغيل للأداة، ستقوم بالفحص الصامت والتثبيت التلقائي للحزم المفقودة التالية:
* `traceroute` و `nmap` (لتحليل وتفكيك البنية التحتية والمنافذ).
* `dnsutils` و `curl` (للاستعلام عن الـ APIs وتحليل النطاقات).
* `python3` (لشحن محرك المقابس المتعدد المسارات وسريع التدفق).
* `netcat (nc)` (لفحص واختبار المنافذ والخدمات).
