# ⚡ DAJ - STRIKE SUITE v2.0 ⚡

أداة متقدمة لتحليل الشبكات وفحص الأمان الرقمي، مصممة خصيصاً لبيئة **Termux** و Linux، تدمج بين سرعة Bash وقوة Python.

---

## 🧬 تفاصيل المشروع
* **المطور**: مازن الدج (Mazen Al-Daj)
* **التوافق**: Termux (Android) & Linux
* **الميزات**: خريطة الشبكة، فحص المنافذ، استخبارات الـ IP، قياس الـ DNS، مراقبة الاستقرار، واختبار الجهد.

---

## 🚀 أمر التثبيت السريع
انسخ الأمر التالي في Terminal:
```bash
pkg install git -y && git clone https://github.com && cd daj-strike && mv daj.sh \(PREFIX/bin/daj && chmod +x\)PREFIX/bin/daj && daj
```

### 🛠️ الإعداد اليدوي
```bash
git clone https://github.com
cd daj-strike
mv daj.sh \$PREFIX/bin/daj
chmod +x \$PREFIX/bin/daj
daj
```

---

## 🧰 المتطلبات (تثبيت تلقائي)
تعتمد الأداة على: `nmap`, `curl`, `python3`, `netcat`.
